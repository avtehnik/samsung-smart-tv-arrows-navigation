samsung-smart-tv-arrows-navigation

Samsung smart tv basic example arrows navigation

Usage 

 var btnsObj = new ButtonsAPI({
        'selector' :'.button',
        'active':'active'
    });
    
    
  btnsObj.navigate('up');
  btnsObj.navigate('left');
